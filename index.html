Project-1: Far Away

    这是一个 React 小型项目。该项目用于 在你旅行出发前，设置好你的待办事项清单，并根据你的清单列表准备情况来给出提示。

    功能说明：

  • 添加 旅行前的待办事项
  • 删除 旅行中不再需要的事项
  • 设置 决定事项是否准备完毕
  • 排序 提供三种方法来给清单中的内容进行排序
  • 重置 清空代办事项列表，重置前给出提示选项

Project-2: React Quiz

    该项目是一个问答项目，点击主页的开始来进行回答题目，答题时设有结束的计时器，共10个题目。

    功能说明：

  • 主页，点击进行问题回答
  • 问题点击后显示 正确的答案
  • 显示进度条，点击 next 进入下一道问题
  • 设有计时器，当时间归零时，进入结算页面
  • 结算页面设有重置按钮，点击后重置应用
